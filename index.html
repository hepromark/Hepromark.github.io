Emotional Cardiography (ECG)

ECG - An application that monitors a patient's emotional state and heart rate using OpenCV, Machine Learning & Arduino sensors.

Uses a Convolutional Neural Network to detect human facial expressions, built with Tensorflow

Streams real-time webcam footage, patient heart rate and model predictions to a Javascript website using Flask

Jesture Bot

A Bluetooth RC Car x Portable Speaker, controlled with hand gestures.

Google's hand recognition model was used to process user hands, left hand for carmovement and right for volume.

A HC-05 Bluetooth module was used to communicate with the Arduino Due, and a L298N motor controller used to control the 4 TT Motors

Satellite Thermal Simulations

Simulated CubeSat satellite models in Siemens NX to investigation chassis performance in space.

Chassis was modeled in SolidWorks, exported to Siemens NX for thermal simulations.

Used surface-to-surface contacts mode to simulate Max Flux and Min Flux cases.
