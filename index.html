Emotional Cardiography (ECG)

ECG aims to alleviate the labor shortage in Canada's medical industry.

It uses a Machine Learning model that I trained and python OpenCV to detect humans emotions using facial expressions.

I used a KY-039 heartbeat sensor and an Arduino to transmit sensor values, incorporating the BPM and likelihood of certain emotions onto the web app

Jesture Bot (ECG)

A Bluetooth RC Car x Portable Speaker, controlled with hand gestures.

Google's hand recognition model was used to process user hands, left hand for carmovement and right for volume. A HC-05 Bluetooth module was used to communicate with the Arduino Due, and a L298N motor controller used to control the 4 TT Motors
